# Обновление: рубрики по дням, вовлечение, петля по статистике, дайджест месяца

Что нового по сравнению с прошлой версией.

## 🗓 Рубрики по дням недели (идея 3)
Каждый день бот собирает подборку под рубрику этого дня:
- Пн — мотивирующий текст
- Вт — кейсы, роботизация
- Ср — Long read (под пост собирается лонгрид в Telegra.ph)
- Чт — обучение, лидерство
- Пт — маркетинг
- Сб — лёгкий пост
- Вс — лайфстайл

Дни новостные (вт–пт) ищут свежак под тему. Авторские (пн, сб, вс) — бот предлагает
оригинальные идеи в нужном стиле. Календарь правится в файле `rubrics.py` обычными словами,
код трогать не нужно.

## 📊 Вовлечение (идея 4)
При сборке поста («✍️ Пост N») бот добавляет идею опроса: вопрос + варианты. Опросы и
комментарии поднимают охват. Отключается переменной `ENGAGEMENT=false`.

## 📈 Петля по статистике (идея 5)
Бот может смотреть, какие НАШИ посты заходят (просмотры/реакции), и смещать утренний
подбор в их сторону. Работает, если настроен доступ к каналу (см. ниже). Тумблер `USE_STATS_LOOP`.

## 📅 Дайджест МЕСЯЦА из наших постов (новый формат идеи 6)
Раз в месяц бот собирает готовый пост «Лучшее из канала» по НАШИМ уже опубликованным
материалам за месяц, отранжированным по охвату и реакциям. Команда для проверки: `/monthly`.

# Внедрение

## Базовая часть (рубрики, вовлечение) — без доп. настройки
1. Распакуй новый архив поверх своей папки (заменятся `bot.py`, `config.py`, `fetcher.py`,
   `formatter.py`, `writer.py`, `requirements.txt`; добавятся `rubrics.py`, `channel_reader.py`,
   `login_telethon.py`).
2. `railway up`
3. Проверь: `/today` → подборка под сегодняшнюю рубрику с кнопками; нажми «Пост 1».
   `/flash` — молнии. Всё это уже работает.

## Доступ к каналу (нужен ТОЛЬКО для `/monthly` и петли по статистике)
Бот не может читать историю канала — нужен вход как пользователь, разово.

1. Зайди на https://my.telegram.org → **API development tools** → создай приложение.
   Получишь **API_ID** (число) и **API_HASH** (строка).
2. Локально на компьютере, в папке проекта:
   ```
   pip install telethon
   python login_telethon.py
   ```
   Введёшь API_ID, API_HASH, свой телефон и код из Telegram. Скрипт напечатает
   длинную строку — это **TELETHON_SESSION**.
3. В Railway → **Variables** добавь четыре переменные:
   ```
   TELETHON_API_ID=<число>
   TELETHON_API_HASH=<строка>
   TELETHON_SESSION=<длинная строка из скрипта>
   CHANNEL_USERNAME=nishchev
   ```
   Railway перезапустится. Теперь `/monthly` соберёт дайджест наших постов, а утренний
   подбор начнёт учитывать статистику.

> Сессия — это доступ к твоему аккаунту, держи строку в секрете (только в Variables, не в гите).

## Настройки (Railway → Variables)
- `MONTHLY_DAY` — день месяца для дайджеста (число или `last`), `MONTHLY_HOUR` — час.
- `ENGAGEMENT` — добавлять опрос (`true`/`false`).
- `USE_STATS_LOOP` — учитывать статистику канала в подборе.
- `FLASH_INTERVAL_HOURS` — частота молний (0 — выключить).
- Рубрики и стили дней — файл `rubrics.py`.

## Команды
`/start` · `/digest` или `/today` (подборка дня) · `/post N` (готовый пост+лонгрид+опрос) ·
`/flash` (эксклюзивы) · `/monthly` (дайджест наших постов).
