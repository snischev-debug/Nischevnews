# Обновление: Молния + Авто-пост + ТОП недели

В боте появились три новых режима. Бот по-прежнему ничего не публикует сам —
всё прилетает тебе в личку готовым к публикации.

## Что нового

**⚡ Молния (режим 1).** Каждые несколько часов бот проверяет свежие ЗАРУБЕЖНЫЕ
материалы, которых ещё нет в РФ-каналах. Если ловит стоящий — присылает короткий
готовый пост в твоём голосе с пометкой «готов к публикации». Публикуешь в три тапа,
пока конкуренты спят. Один и тот же сюжет повторно не пришлёт.

**✍️ Авто-пост + лонгрид (режим 2).** Под утренней подборкой теперь кнопки
«✍️ Пост 1…5». Жмёшь — бот пишет развёрнутый пост для канала и собирает лонгрид
в Telegra.ph, присылает текст + готовую ссылку. То же делает команда `/post N`.

**🏆 ТОП недели (режим 6).** Раз в неделю (по умолчанию пятница 17:00 МСК) бот
присылает готовый пост-дайджест недели — публикуемый, не совет.

## Новые команды
- `/post N` — полный пост + лонгрид по теме N из последней подборки
- `/flash` — проверить эксклюзивы прямо сейчас
- `/weekly` — собрать «ТОП недели» сейчас
(старые `/start`, `/digest` на месте)

## Как внедрить (3 шага)

1. **Замени файлы.** Скачай новый архив и распакуй поверх своей папки (или просто
   замени изменённые файлы: `bot.py`, `config.py`, `fetcher.py`, `formatter.py`,
   `requirements.txt`, и добавь новые: `writer.py`, `telegraph.py`, `seen_store.py`).

2. **Залей обновление на Railway.** В терминале, находясь в папке проекта:
   ```
   railway up
   ```
   Railway пересоберёт сервис с новыми библиотеками (добавился aiohttp).

3. **Проверь.** Напиши боту:
   - `/digest` → придёт подборка с кнопками «✍️ Пост N». Нажми одну — получишь
     пост + ссылку на лонгрид.
   - `/flash` → проверка эксклюзивов сейчас.
   - `/weekly` → сборка ТОП недели сейчас.

Готово. Дальше всё работает по расписанию само.

## Настройки (Railway → Variables, без правки кода)
- `FLASH_INTERVAL_HOURS` — как часто искать эксклюзивы (по умолчанию 4 ч). `0` — выключить молнию.
- `WEEKLY_DAY` / `WEEKLY_HOUR` — день и час «ТОП недели» (по умолчанию `fri` 17:00).
- `AUTHOR_NAME` / `CHANNEL_URL` — подпись и ссылка в лонгридах Telegra.ph.
- `MODEL` — `claude-opus-4-8` (качество) или `claude-haiku-4-5-20251001` (экономия).

## Тонкости
- Лонгриды публикуются в Telegra.ph через API и сразу доступны по ссылке. Редактировать
  их в веб-редакторе Telegra.ph нельзя (страница создана ботом) — но можно перегенерировать
  через бота или попросить меня собрать конкретный лонгрид отдельно.
- Стоимость с новыми режимами всё ещё небольшая: молния — 1 короткий запрос за скан,
  пост/лонгрид — по запросу, недельный — раз в неделю. Лимит расходов в консоли Anthropic
  никто не отменял, держи его включённым.
- Память «уже отправленного» (seen.json) на Railway сбрасывается при передеплое —
  максимум одна повторная молния после обновления, это нормально.
