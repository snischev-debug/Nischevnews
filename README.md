# Утренний контент-советник для канала @nishchev

Бот **ничего не публикует** в твой канал. Каждое утро он присылает **лично тебе**
в Telegram подборку свежих крутых тем (лидерство, маркетинг, цифровая трансформация
предприятий, роботизация) с советом, как обыграть каждую в посте. Свежак из-за рубежа,
которого ещё нет в РФ-каналах, помечает 🌍 **эксклюзив**.

Мозг — Claude Opus 4.8. Доставка — обычный Telegram-бот.

---

## Что приходит каждое утро

Сообщение в личку: вступление + 5 тем. По каждой — суть своими словами,
**💡 как обыграть** (готовый угол и хук для поста), **🎯 почему зайдёт аудитории**,
ссылка на источник и метка 🌍 для эксклюзива. Дальше ты сам решаешь, о чём писать.

# Запуск: по шагам, без спешки

## Шаг 1. Сделать бота в BotFather (5 минут)

1. Открой [@BotFather](https://t.me/BotFather), отправь `/newbot`.
2. Придумай имя и username (username должен заканчиваться на `bot`).
3. BotFather пришлёт **токен** — длинная строка вида `8123456789:AAH...`. Сохрани её.
4. Найди только что созданного бота в поиске Telegram и нажми **«Запустить» / Start**
   (или отправь ему любое сообщение). Без этого бот не сможет писать тебе в личку.

Канал @nishchev трогать не нужно — бот в него не пишет.

---

## Шаг 2. Что такое `.env` и что туда вписать

`.env` — это простой текстовый файл с тремя твоими «ключами». В папке проекта уже лежит
образец `.env.example`. Скопируй его в файл с именем `.env` и впиши три значения:

| Строка | Что это | Где взять |
|---|---|---|
| `BOT_TOKEN` | токен бота | из BotFather (шаг 1) |
| `ADMIN_CHAT_ID` | твой личный числовой ID в Telegram | напиши [@userinfobot](https://t.me/userinfobot) — он сразу пришлёт твой `Id` (число) |
| `ANTHROPIC_API_KEY` | ключ к Claude | console.anthropic.com → раздел **API Keys** → Create Key |

Готовый `.env` выглядит так (вместо троеточий — твои значения):

```
BOT_TOKEN=8123456789:AAH...твой_токен...
ADMIN_CHAT_ID=123456789
ANTHROPIC_API_KEY=sk-ant-...твой_ключ...
```

Остальные строки в файле (модель, время рассылки) уже заданы по умолчанию —
менять не обязательно. Утро по умолчанию — **09:00 МСК**; чтобы сдвинуть, поменяй
`DIGEST_HOUR`.

## Шаг 3. Запустить программу

«Запустить» = включить программу, которая и будет слать тебе советы. Нужен компьютер
с интернетом. Команды вводятся в терминале (на Mac — «Терминал», на Windows — «PowerShell»),
находясь в папке проекта.

```bash
# 1) создать изолированное окружение (делается один раз)
python3 -m venv .venv

# 2) включить его
source .venv/bin/activate        # Windows: .venv\Scripts\activate

# 3) поставить нужные библиотеки (один раз)
pip install -r requirements.txt

# 4) запустить бота
python bot.py
```

Теперь напиши своему боту в Telegram `/start` — он ответит. Команда `/digest` —
собрать подборку прямо сейчас, не дожидаясь утра. Если всё хорошо — через минуту
придёт подборка.

> Важно: пока окно с `python bot.py` открыто и компьютер включён — бот работает и
> присылает советы каждое утро. Закрыл окно / выключил комп — бот молчит.

## Чтобы советы приходили КАЖДОЕ утро сами (always-on)

Чтобы не держать компьютер включённым, программу ставят на «всегда включённый» сервер.
Самый простой путь без своего сервера:

1. Зарегистрируйся на [railway.app](https://railway.app) (есть бесплатный старт).
2. Создай новый проект → загрузи эту папку (или подключи репозиторий).
3. В разделе **Variables** впиши те же три значения, что в `.env`
   (`BOT_TOKEN`, `ADMIN_CHAT_ID`, `ANTHROPIC_API_KEY`).
4. Команда запуска: `python bot.py`. Готово — бот живёт в облаке и пишет тебе каждое утро.

Если шаг с сервером кажется муторным — это ровно та задача, которую быстро добьёт
**Claude Code**: открываешь папку, говоришь «подними этот бот на Railway» — и он
проведёт по шагам.

## Как менять поведение

- **Темы, твой голос, запреты на штампы** — файл `audience.md`.
- **Источники** (добавить канал/издание) — файл `sources.yaml`.
  TG-канал добавляется строкой `https://rsshub.app/telegram/channel/<username>`.
- **Время утренней рассылки** — `DIGEST_HOUR` в `.env`.
- Чтение TG-каналов идёт через публичный RSSHub и иногда лимитится; издания (HBR,
  IEEE и пр.) работают стабильнее и как раз дают эксклюзив.
